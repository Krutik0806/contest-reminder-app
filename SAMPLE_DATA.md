## Tips for Real Data

### Where to find upcoming contests:
1. **Clist.by** - https://clist.by/
   - Aggregates contests from all platforms
   - Use their API for automatic updates (future enhancement)

2. **Platform Websites:**
   - Codeforces: https://codeforces.com/contests
   - CodeChef: https://www.codechef.com/contests
   - LeetCode: https://leetcode.com/contest
   - AtCoder: https://atcoder.jp/contests

3. **Contest Calendars:**
   - Google "competitive programming calendar"
   - Check Reddit r/competitiveprogramming

### Best Practices:
- Add contests at least 24 hours in advance
- Use accurate start times
- Double-check timezone conversions
- Include direct contest registration links
- Add relevant tags for filtering
